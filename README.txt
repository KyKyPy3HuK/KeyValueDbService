Руководство пользователя

Папка KeyValueDbService содержит в себе исходный код основный прграммы-сервиса и руководство по его запуску.

Папка KVDSClient содержит исходный код дополнительной програмы-примера которя использует класс-библиотеку для использования сервиса и руоводство по его запуску.

Файл KVDSConnectuon.java представляет собой класс-библиотеку для боллее удобного общения с сервисом из java кода.

Файл key_value_db_service_api.yml является спецификацией OpenAPI для сервиса.
